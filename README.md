## Install

```sh
npm install
```

## Usage

- Run web server

```sh
npm run start
```

- Run local blockchain

```sh
npm run ganache
```

- Deploy NFT contract

```sh
npx hardhat run scripts/deploy.js --network ganache
```

## Run tests

```sh
npx hardhat test
```
